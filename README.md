# HYPEFest
## A web app for browsing, reviewing, and listening to music festivals

### Built with:
- Ruby on Rails
- PosgreSQL
- Sorcery Authentication GEM
- Bootstrap
- Pure CSS
- Google Map API
- Facebook API
- Embedded SoundCloud Player
